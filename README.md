# Pedido de Perdão — Projeto

Pequeno projeto de landing page romântica para pedido de perdão.

Arquivos principais:
- `index.html`
- `styles.css`
- `script.js`
- `assets/` — imagens e placeholders

Como personalizar:
- Substitua as imagens em `assets/` por fotos reais (`photo-left.svg`, `photo-right.svg`, `photo1..4.svg`).
- Coloque um ficheiro de música em `assets/music.mp3` e atualize o `src` do elemento `<audio>` em `index.html`.
- Ajuste a `startDate` em `script.js` para a data real do início do relacionamento.
- Para gerar PDF da carta, abra a secção "Carta aberta" e use imprimir (Ctrl+P) ou usar funções do navegador.

Abrir: carregar `index.html` no navegador.
